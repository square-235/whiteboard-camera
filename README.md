# 本地白板应用

一个基于Web的本地白板应用，支持绘图、摄像头等功能，无需服务器即可运行。

## 功能特性

- 本地绘图功能（画笔和橡皮擦）
- 画布移动、缩放
- 摄像头画面插入与控制
- 保存内容为图片
- 响应式设计，支持移动设备

## 运行环境要求

- 现代浏览器（支持Canvas API和摄像头API）

## 安装和运行

1. **下载或克隆项目代码**

2. **打开应用**：
   直接用浏览器打开 `public/index.html` 文件即可使用

## 使用说明

1. **绘图工具**：
   - 画笔：用于绘制
   - 橡皮擦：用于擦除
   - 移动：用于移动画布（也可按住空格键临时切换）

2. **画布操作**：
   - 鼠标滚轮：缩放画布
   - 拖拽：移动画布
   - +/-按钮：精确控制缩放

3. **摄像头功能**：
   - 点击"开启摄像头"按钮启动摄像头
   - 拖拽摄像头画面可移动位置
   - "冻结画面"按钮可暂停摄像头更新

4. **其他功能**：
   - 清空白板：清除所有绘图内容
   - 保存为图片：将当前画布保存为PNG图片

## 技术栈

- 前端：Vue.js (通过CDN引入)
- 绘图：HTML5 Canvas
- 摄像头：WebRTC API

## 常见问题及解决方案

### 1. 移动和缩放功能无法正常工作
- 确保使用现代浏览器（Chrome、Firefox、Edge等）
- 检查浏览器是否禁用了JavaScript

### 2. 摄像头无法启动或提示权限问题
- 确保浏览器有摄像头访问权限：
  - Chrome：设置 → 隐私设置和安全性 → 网站设置 → 摄像头
  - Firefox：设置 → 隐私与安全 → 权限 → 摄像头
- 确保电脑已连接摄像头设备
- 某些浏览器在本地文件协议(file://)下可能限制摄像头访问，建议使用HTTP服务器运行

### 3. 画布内容显示异常
- 刷新页面尝试重新初始化画布
- 检查浏览器控制台是否有错误信息

## 注意事项

- 应用完全在本地运行，不上传任何数据到外部服务器
- 摄像头功能需要用户授权访问
- 画布内容不会自动保存，需要手动点击"保存为图片"按钮保存
