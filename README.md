# 基于Web的白板和视频展台应用

一个基于Web的白板和视频展台应用，支持绘图、摄像头等功能。
使用TRAE AI构建

## 功能特性

- 绘图功能（画笔和橡皮擦）
- 画布移动、缩放
- 白板/视频展台二合一，快捷切换
- 保存内容为图片
- 响应式设计，支持移动设备

## 运行环境要求

- 现代浏览器（支持Canvas API和摄像头API）

## 安装和运行

1. **下载或克隆项目代码**

2. **打开应用**：
   直接用浏览器打开 `public/index.html` 文件即可使用

## 技术栈

- 前端：Vue.js (通过CDN引入)
- 绘图：HTML5 Canvas
- 摄像头：WebRTC API

## 常见问题及解决方案

### 1. 摄像头无法启动或提示权限问题
- 确保浏览器有摄像头访问权限：
  - Chrome：设置 → 隐私设置和安全性 → 网站设置 → 摄像头
  - Firefox：设置 → 隐私与安全 → 权限 → 摄像头
- 确保电脑已连接摄像头设备
- 某些浏览器在本地文件协议(file://)下可能限制摄像头访问，推荐Edge、Firefox等主流浏览器
